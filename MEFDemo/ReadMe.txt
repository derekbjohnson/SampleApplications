This solution is the source code to a talk I presented to the Northeast Arkansas .NET Users Group (http://neadnug.com/).

Do note that in order to build the MEFPrism project, you will need the binary files for Prism version 4.  More information here: http://msdn.microsoft.com/en-us/library/gg406140.aspx

In order to run the MEFConsole samples, you can edit the top line with "#define IncludeThis".  Only one of the programs should be active at any one time.
